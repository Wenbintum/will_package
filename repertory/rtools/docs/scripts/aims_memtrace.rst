aims_memtrace.py
================

Small script to extract the peak memory allocation from the memtrace_000 output files that are generated by FHI-aims when compiled with the flag *memtrace_use TRUE*.

Usage:
""""""

>>> memtrace.py memtrace-000

Maximum memory allocation was: 21.9MB (raw: 23011756 bytes)
Max size was hit in line 4105: 
['02/10/2014', '14:16:23:623', 'ALLOC', '16', '23011756', '23011756', 'hartree_atoms', '[subroutine', 'hartree_potential_real_coeff', '@', 'hartree_potential_real_p0.f90{112-112}]']
